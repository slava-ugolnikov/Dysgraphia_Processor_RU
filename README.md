# CV Dysgraphia. Rule-based разметка дисграфических ошибок

Система для **категоризации дисграфических ошибок** с помощью набора лингвистических правил (без ML).
Стек включает лингвистические модули и генерацию итогового **Excel** с разметкой ошибок.

---

## Возможности проекта

### 1. Лингвистическая обработка

Используются:

* `pymorphy2`

  
### 3. Правила выявления ошибок (rule-based)

Поддерживаются типы:

* **Пропуски** букв/слогов/слов
* **Перестановки**
* **Персеверации**
* **Антиципации**
* **Вставки букв**
* **Смешения**:

  * звонкие/глухие
  * свистящие/шипящие
  * аффрикаты
  * о/у, ё/ю
 
* **Ошибки мягкости** (ь, йотированные гласные)
* **Ошибки со звуком Й**
* **Пропуски разделительных знаков**

### 4. Экспорт разметки в Excel

Формат итоговой таблицы:

| Задуманное слово | Написанное слово | Тип ошибки |
