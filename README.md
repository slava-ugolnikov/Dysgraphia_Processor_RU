# CV Dysgraphia. Rule-based разметка дисграфических ошибок

Система для **категоризации дисграфических ошибок** с помощью набора лингвистических правил (без ML).
Стек включает лингвистические модули и генерацию итогового **Excel** с разметкой ошибок.

---

## Возможности проекта

### 1. Лингвистическая обработка

Используются:

* `pymorphy2`

  
### 3. Правила выявления ошибок (rule-based)

Поддерживаются типы:

* **Пропуски** букв/слогов/слов
* **Перестановки**
* **Персеверации**
* **Антиципации**
* **Вставки букв**
* **Смешения**:

  * звонкие/глухие
  * свистящие/шипящие
  * аффрикаты
  * о/у, ё/ю
 
* **Ошибки мягкости** (ь, йотированные гласные)
* **Ошибки со звуком Й**
* **Пропуски разделительных знаков**

### 4. Экспорт разметки в Excel

## Установка

## Теоретическая база
Категоризация ошибок основана на классификации, предложенной Р.И. Лалаевой (1989).

Обращаем внимание, что программа может допускать ошибки и нуждается в дальнейшем совершенствовании. Также проект работает только с Python3.10, новые версии не поддерживаются... По крайней мере у меня на компьютере
